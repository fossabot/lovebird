LoveBird

a PHP/mySQL web app with numerous modules for building a web site

Get the code: Github

LoveBird includes modules for a blog, an online store, a photographer site, a file upload tool and more.

Features

  1. Blog Engine
  2. Photographer Tools
  3. Online Store
  4. File Manager
  5. Page Builder
  6. Stock Quote [via API]
  7. Messages Feature
  8. Send Money: View
